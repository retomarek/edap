# Getting Started

This part helps you set up everything you need to follow the examples in this book.

```{admonition} What you will learn
:class: tip
- How to install Python and choose an IDE
- How to work with Jupyter Notebooks
- How to install and manage Python packages
```

## Overview

Before diving into data analysis, you need a working Python environment. The following sections guide you through:

1. **Install Python** — Install Python 3.12+ and choose an IDE (VS Code or PyCharm)
2. **Jupyter Notebooks** — Learn the basics of working with Jupyter Notebooks
3. **Packages** — Install the required packages using pip and virtual environments

If you already have a Python environment set up, you can skip ahead to the [Python Basics](pythonBasics) chapter.
