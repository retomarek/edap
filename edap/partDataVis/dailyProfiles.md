# Daily Profiles

Daily profile visualizations show how a variable changes over the course of a day. They are essential for understanding usage patterns, occupancy behavior, and system operation.

## Recipes

- **Overview** — Display all daily profiles as a compact overview
- **Overlayed** — Overlay daily profiles colored by weekday, month, or season
- **Mean Profile** — Show the average daily profile with confidence bands
- **Decomposed** — Separate the daily profile into trend and residual components
